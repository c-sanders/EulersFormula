About this Package.

At the heart of this package lie two files - which when taken together, implement a C++ class. The class that they implement provides an implementation of Euler's formula.

Euler's formula can be calculated in a number of different ways, but this class uses the Power Series definition to calculate its values.

The class.

The class is implemented by the following two files;
  • include/Euler.hpp
  • src/Euler.cpp
The first file contains the declaration of the class and it can be found in the include sub-directory of the package. The second file contains the definition of the class and it can be found in the src sub-directory of the package.

Class test program.

A third file provides an illustration of how to use the class. This file is called e.cpp, and it can also be found in the src sub-directory of the package.
