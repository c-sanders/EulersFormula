Dependencies for this Package.

Both the class itself and the class test program have dependencies on external software libraries.

The class.

The class depends only on the GNU Scientific Library (GSL).

In particular, it makes use of the following mathematical Factorial function which is provided by the GSL;

double gsl_sf_fact(usigned int n)

where it is used in the calculation for each of the values of the components for Euler's formula.

The class test program.

The class test program depends only on the Boost library.

In particular, it makes use of the following Lexical Cast function which is provided by the Boost library;

boost::lexical_cast(arg String)

Implication of dependencies.

These dependencies on external software libraries come with one big implication, and that is;

The system upon which the package is being configured, built, and installed, must have an appropriate version of both the GSL and the Boost Library installed on it.

Thus, the person who is responsible for configuring, building, and installing this package, must ensure that appropriate versions of both of these two dependent libraries are installed on the system. Furthermore, they may even need to know exactly where both of these libraries are installed on the system.


The next page discusses how to configure this package so that both the GSL and the Boost Library can be used by the build process.
