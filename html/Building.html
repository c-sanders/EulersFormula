Building and installing this Package.

Once this package has been successfully configured, the next step is to build and then install it.

Building the package.

Invoke the following command from within the directory in which the configure script was run;

make

If this command runs successfully, then it should have built the test program.

Installing the packge.

To install this test program, along with the two files that implement the class, invoke the following command from within the directory in which the configure script was run;

make install
